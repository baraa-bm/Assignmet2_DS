# Advanced B-Tree / 2-3-4 Tree Assignment

## Project Description

This project implements a 2-3-4 Tree, which is a B-Tree of order 4.  
The tree supports insertion, search, level-order printing, file-based input, logging, split counting, and save/restore functionality.

## Files

- `src/main.cpp`: main implementation
- `input.txt`: operations file
- `output.txt`: generated output file
- `log.txt`: generated log file
- `snapshot.dat`: generated save file
- `README.md`: project explanation

## How to Run

```bash
g++ src/main.cpp -o btree
./btree
